🍴 FoodSwap – Plate Trading Platform
📌 Overview
FoodSwap is a web application designed to allow cafeteria users to exchange parts of their meals (entrée, main dish, dessert, side).
The goal is to improve user satisfaction, reduce food waste, and create a more social dining experience.

🚀 Features
🔐 Secure Authentication (Laravel Policies & Gates)

👤 User Profiles (preferences, trade history)

🍽️ Offer & Request System (create, update, delete trades)

🔄 Trade Matching (connect users who want to swap components)

📊 Statistics Dashboard (most traded items, user activity)

🛠️ Admin Panel (manage users, monitor trades, handle reports)

📱 Responsive UI (Tailwind CSS, mobile‑friendly design)
